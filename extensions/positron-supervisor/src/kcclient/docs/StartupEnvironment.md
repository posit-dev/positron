# StartupEnvironment

The kernel\'s startup environment: \'none\' for normal startup, \'shell\' for a login shell, \'command\' for a preflight command, \'script\' to run a script. Only relevant on POSIX-like systems.

## Enum

* `None` (value: `'none'`)

* `Shell` (value: `'shell'`)

* `Command` (value: `'command'`)

* `Script` (value: `'script'`)
