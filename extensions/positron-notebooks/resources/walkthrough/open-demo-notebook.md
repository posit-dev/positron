We've prepared a working example so you can see the Positron Notebook Editor in action easily.

#### Steps:
1. **Create a new folder from Git** by running the [Workspaces: New Folder from Git](command:positron.workbench.action.newFolderFromGit) command in the Command Palette.

2. **Paste the demo URL**: `https://github.com/posit-dev/positron-demos-notebooks`

3. **Open the notebook**: Once cloned, open `world-happiness-report.ipynb`.

4. **Verify the Positron Notebook Editor is enabled**:

If you do not see the **Run All Cells** button as the first item in the Notebook navigation bar, you may need to [enable Positron Notebook](command:workbench.action.openSettings?positron.notebook.enabled?true).
