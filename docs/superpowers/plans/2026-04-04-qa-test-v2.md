# QA Test v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Reduce AI-driven QA testing from ~15 tool calls / 20+ seconds to 4 tool calls / <30 seconds (excluding app launch) by adding a `/run-plan` endpoint, enriched observer, state reset, and POM reference generator.

**Architecture:** Single app launch via explore runner. AI reads a generated POM reference for type-safe method signatures, sends the entire test as one `POST /run-plan` call, and gets a structured report back. State reset between retries. Explore mode fallback for adaptive debugging.

**Tech Stack:** TypeScript, Node.js HTTP server, Playwright test infrastructure, existing Page Object Models.

**Spec:** `docs/superpowers/specs/2026-04-03-qa-test-v2-design.md`

**IMPORTANT:** This project uses tabs for indentation. All TypeScript files must use tabs. Run `npm run precommit -- <file>` to verify before committing. Use `npm run build-check` to verify compilation (NEVER run `tsc` directly).

/*---------------------------------------------------------------------------------------------
 *  Copyright (C) 2026 Posit Software, PBC. All rights reserved.
 *  Licensed under the Elastic License 2.0. See LICENSE.txt for license information.
 *--------------------------------------------------------------------------------------------*/

import { Application } from '../../infra/application';

const RESET_STEP_TIMEOUT = 2000;
const SESSION_RESET_TIMEOUT = 6000;

/**
 * Attempt to resolve a Promise within a timeout.
 * Returns true if the operation completed, false if it timed out.
 */
async function withTimeout(fn: () => Promise<void>, timeout: number): Promise<boolean> {
	try {
		await Promise.race([
			fn(),
			new Promise<never>((_, reject) =>
				setTimeout(() => reject(new Error('reset step timeout')), timeout)
			),
		]);
		return true;
	} catch {
		return false;
	}
}

/**
 * Best-effort state cleanup between retry attempts.
 *
 * Each step has a short timeout and swallows errors -- a failing cleanup step
 * must never block the retry. Returns a list of actions taken (or skipped).
 */
export async function resetState(app: Application): Promise<string[]> {
	const actions: string[] = [];
	const page = app.code.driver.page;

	// 1. Dismiss blocking UI (dialogs, quick input, context menus)
	try {
		for (let i = 0; i < 3; i++) {
			await page.keyboard.press('Escape');
			await page.waitForTimeout(100);
		}
		actions.push('Dismissed overlays (3x Escape)');
	} catch {
		actions.push('Dismiss overlays: skipped');
	}

	// 2. Close all open editors
	const closedEditors = await withTimeout(async () => {
		await app.workbench.quickaccess.runCommand('workbench.action.closeAllEditors');
	}, RESET_STEP_TIMEOUT);
	actions.push(closedEditors ? 'Closed all editors' : 'Close editors: skipped');

	// 3. Clear notifications
	const clearedNotifs = await withTimeout(async () => {
		await app.workbench.quickaccess.runCommand('notifications.clearAll');
	}, RESET_STEP_TIMEOUT);
	actions.push(clearedNotifs ? 'Cleared notifications' : 'Clear notifications: skipped');

	// 4. Delete all sessions (needs longer timeout -- involves UI)
	const deletedSessions = await withTimeout(async () => {
		await app.workbench.sessions.deleteAll();
	}, SESSION_RESET_TIMEOUT);
	actions.push(deletedSessions ? 'Deleted all sessions' : 'Delete sessions: skipped');

	// 5. Ensure bottom panel is visible (console)
	const modifier = process.platform === 'darwin' ? 'Meta' : 'Control';
	try {
		await page.keyboard.press(`${modifier}+j`);
		await page.waitForTimeout(200);
		actions.push('Toggled bottom panel');
	} catch {
		actions.push('Toggle panel: skipped');
	}

	// 6. Focus editor area
	const focusedEditor = await withTimeout(async () => {
		await app.workbench.quickaccess.runCommand('workbench.action.focusActiveEditorGroup');
	}, RESET_STEP_TIMEOUT);
	actions.push(focusedEditor ? 'Focused editor area' : 'Focus editor: skipped');

	return actions;
}
```

// ---------------------------------------------------------------------------
//  /run-plan executor (full test plan with timeouts and reset)
// ---------------------------------------------------------------------------

/**
 * Execute a complete test plan in one call.
 *
 * Differences from /batch:
 *   - Optional state reset before execution (resetBefore)
 *   - Per-step timeout with a configurable default (stepTimeout)
 *   - Structured RunPlanResult with passed/failed counts
 *   - Fail-fast: stops at first failure, observes state, returns report
 */
export async function executeRunPlan(app: Application, request: RunPlanRequest): Promise<RunPlanResult> {
	const planLabel = request.title || 'Run Plan';
	const startTime = Date.now();

	return await test.step(planLabel, async () => {
		// 1. Optional state reset
		let resetActions: string[] | undefined;
		if (request.resetBefore) {
			resetActions = await test.step('State reset', async () => {
				return await resetState(app);
			});
		}

		// 2. Execute steps sequentially with per-step timeouts
		const stepResults: RunPlanStepResult[] = [];
		const defaultTimeout = request.stepTimeout ?? 10000;

		for (let i = 0; i < request.steps.length; i++) {
			const step = request.steps[i];
			const timeout = step.timeout ?? defaultTimeout;
			const stepStart = Date.now();
			const title = step.title || `Step ${i + 1}`;

			let actionResult: ActionResult;
			try {
				actionResult = await Promise.race([
					step.type === 'pom'
						? executePomDirect(app, step)
						: executeActionDirect(app, step),
					createStepTimeout(timeout),
				]);
			} catch (err: any) {
				// Thrown error: timeout or POM method failure
				stepResults.push({
					title,
					success: false,
					error: err.message ?? String(err),
					duration: Date.now() - stepStart,
				});
				const state = await observeState(app);
				return {
					passed: stepResults.filter(s => s.success).length,
					failed: 1,
					steps: stepResults,
					skipped: request.steps.length - i - 1,
					totalDuration: Date.now() - startTime,
					state,
					resetActions,
				};
			}

			// Returned error (e.g., unknown POM or method)
			if (!actionResult.success) {
				stepResults.push({
					title,
					success: false,
					error: actionResult.error,
					duration: Date.now() - stepStart,
				});
				const state = await observeState(app);
				return {
					passed: stepResults.filter(s => s.success).length,
					failed: 1,
					steps: stepResults,
					skipped: request.steps.length - i - 1,
					totalDuration: Date.now() - startTime,
					state,
					resetActions,
				};
			}

			stepResults.push({
				title,
				success: true,
				duration: Date.now() - stepStart,
			});
		}

		// 3. All steps passed -- observe final state
		const state = await observeState(app);
		return {
			passed: stepResults.length,
			failed: 0,
			steps: stepResults,
			totalDuration: Date.now() - startTime,
			state,
			resetActions,
		};
	});
}

/*---------------------------------------------------------------------------------------------
 *  Copyright (C) 2026 Posit Software, PBC. All rights reserved.
 *  Licensed under the Elastic License 2.0. See LICENSE.txt for license information.
 *--------------------------------------------------------------------------------------------*/

/**
 * Generate a compact POM reference file with full TypeScript type signatures.
 *
 * Parses the Workbench class and each POM source file to extract public method
 * signatures including union types, optional parameters, and defaults.
 * Also enumerates getter-based sub-objects (e.g., dataExplorer.grid).
 *
 * Output: test/e2e/tests/qa-generated/pom-reference.md
 *
 * Usage:
 *   npx tsx scripts/generate-pom-reference.ts
 */

import * as fs from 'fs';
import * as path from 'path';

const E2E_ROOT = path.resolve(__dirname, '../test/e2e');
const WORKBENCH_PATH = path.join(E2E_ROOT, 'infra/workbench.ts');
const OUTPUT_DIR = path.join(E2E_ROOT, 'tests/qa-generated');
const OUTPUT_PATH = path.join(OUTPUT_DIR, 'pom-reference.md');

// ---------------------------------------------------------------------------
// 1. Parse Workbench class: POM name -> TypeName -> import path
// ---------------------------------------------------------------------------

interface PomEntry {
	name: string;
	typeName: string;
	filePath: string;
}

function parseWorkbench(): PomEntry[] {
	const source = fs.readFileSync(WORKBENCH_PATH, 'utf-8');

	// Extract imports: import { TypeName } from './path'
	const importMap = new Map<string, string>();
	for (const match of source.matchAll(/import\s+\{([^}]+)\}\s+from\s+['"]([^'"]+)['"]/g)) {
		const types = match[1].split(',').map(t => t.trim());
		const importPath = match[2];
		for (const type of types) {
			importMap.set(type, importPath);
		}
	}

	// Extract readonly properties: readonly name: TypeName
	const entries: PomEntry[] = [];
	for (const match of source.matchAll(/readonly\s+(\w+)\s*:\s*(\w+)/g)) {
		const name = match[1];
		const typeName = match[2];
		const importPath = importMap.get(typeName);
		if (!importPath) { continue; }

		// Resolve file path relative to infra/
		let filePath = path.resolve(path.join(E2E_ROOT, 'infra'), importPath);
		if (!filePath.endsWith('.ts') && !filePath.endsWith('.js')) {
			filePath += '.ts';
		}
		if (filePath.endsWith('.js')) {
			filePath = filePath.replace(/\.js$/, '.ts');
		}

		if (fs.existsSync(filePath)) {
			entries.push({ name, typeName, filePath });
		}
	}

	return entries;
}

// ---------------------------------------------------------------------------
// 2. Extract method signatures from a POM source file
// ---------------------------------------------------------------------------

function findMatchingBracket(text: string, startIndex: number, open: string, close: string): number {
	let depth = 0;
	for (let i = startIndex; i < text.length; i++) {
		if (text[i] === open) { depth++; }
		if (text[i] === close) { depth--; if (depth === 0) { return i; } }
	}
	return -1;
}

function extractMethodSignatures(source: string): string[] {
	const signatures: string[] = [];
	const lines = source.split('\n');

	for (let i = 0; i < lines.length; i++) {
		const line = lines[i];

		// Skip private/protected methods
		if (/^\t(?:private|protected)\s+/.test(line)) { continue; }

		// Match method declarations: [public] [async] methodName( or methodName<
		const methodMatch = line.match(/^\t(?:public\s+)?(?:async\s+)?(\w+)\s*[<(]/);
		if (!methodMatch) { continue; }

		const name = methodMatch[1];
		if (name === 'constructor' || name.startsWith('_')) { continue; }

		// Collect lines until we find the opening brace '{' or semicolon ';'
		let fullSig = '';
		for (let j = i; j < Math.min(i + 15, lines.length); j++) {
			fullSig += (j === i ? '' : ' ') + lines[j];
			// Stop at body start or statement end
			if (/\{[\s]*$/.test(lines[j]) || /;\s*$/.test(lines[j])) { break; }
		}

		// Extract everything from method name to before '{'
		const bodyStart = fullSig.search(/\s*\{[\s]*$/);
		const sigText = bodyStart !== -1 ? fullSig.substring(0, bodyStart) : fullSig;

		// Clean up: remove leading tab/whitespace, access modifiers
		let cleaned = sigText
			.replace(/^\s+/, '')
			.replace(/^public\s+/, '')
			.replace(/^async\s+/, 'async ')
			.replace(/\s+/g, ' ')
			.trim();

		// Remove 'async ' prefix -- we show it as return type instead
		cleaned = cleaned.replace(/^async\s+/, '');

		if (!cleaned.includes('(')) { continue; }

		// Normalize: ensure we have return type annotation
		// If it ends with ) and no :, it returns void
		if (/\)\s*$/.test(cleaned) && !cleaned.includes('): ')) {
			// Check if there's a ): pattern somewhere
			const lastParen = cleaned.lastIndexOf(')');
			const afterParen = cleaned.substring(lastParen + 1).trim();
			if (!afterParen.startsWith(':')) {
				cleaned += ': void';
			}
		}

		signatures.push('- ' + cleaned);
	}

	return signatures;
}

// ---------------------------------------------------------------------------
// 3. Extract getter-based sub-objects
// ---------------------------------------------------------------------------

interface SubObject {
	parentName: string;
	getterName: string;
	returnType: string;
	filePath: string;
}

function findGetterSubObjects(source: string, pomName: string, allImports: Map<string, string>): SubObject[] {
	const subObjects: SubObject[] = [];

	// Match: get propertyName(): ReturnType {
	for (const match of source.matchAll(/^\tget\s+(\w+)\(\)\s*:\s*(\w+)/gm)) {
		const getterName = match[1];
		const returnType = match[2];

		if (getterName.startsWith('_')) { continue; }

		// Find the source file for the return type
		const importPath = allImports.get(returnType);
		if (!importPath) { continue; }

		let filePath = path.resolve(path.dirname(allImports.get('__self__') || ''), importPath);
		if (!filePath.endsWith('.ts')) { filePath += '.ts'; }
		if (filePath.endsWith('.js')) { filePath = filePath.replace(/\.js$/, '.ts'); }

		if (fs.existsSync(filePath)) {
			subObjects.push({ parentName: pomName, getterName, returnType, filePath });
		}
	}

	return subObjects;
}

function parseImportsFromSource(source: string): Map<string, string> {
	const imports = new Map<string, string>();
	for (const match of source.matchAll(/import\s+\{([^}]+)\}\s+from\s+['"]([^'"]+)['"]/g)) {
		const types = match[1].split(',').map(t => t.trim());
		for (const type of types) {
			imports.set(type, match[2]);
		}
	}
	return imports;
}

// ---------------------------------------------------------------------------
// 4. Generate the reference file
// ---------------------------------------------------------------------------

function generate(): void {
	const entries = parseWorkbench();

	const lines: string[] = [
		'# POM Reference',
		'',
		'Auto-generated from POM source files. Do not edit manually.',
		`Generated: ${new Date().toISOString().split('T')[0]}`,
		'',
		'Usage: Read this file to find correct method names, parameter types,',
		'and return types before generating test code or /run-plan steps.',
		'',
		`Total POMs: ${entries.length}`,
		'',
		'---',
		'',
	];

	for (const entry of entries) {
		const source = fs.readFileSync(entry.filePath, 'utf-8');
		const relativePath = path.relative(path.resolve(__dirname, '..'), entry.filePath);
		const signatures = extractMethodSignatures(source);

		if (signatures.length === 0) { continue; }

		lines.push(`## ${entry.name} (${relativePath})`);
		lines.push('');
		lines.push(...signatures);
		lines.push('');

		// Check for getter-based sub-objects
		const imports = parseImportsFromSource(source);
		// Store the source file path for relative import resolution
		imports.set('__self__', entry.filePath);

		const subObjects = findGetterSubObjects(source, entry.name, imports);
		for (const sub of subObjects) {
			if (!fs.existsSync(sub.filePath)) { continue; }
			const subSource = fs.readFileSync(sub.filePath, 'utf-8');
			const subSigs = extractMethodSignatures(subSource);
			if (subSigs.length === 0) { continue; }

			const subRelative = path.relative(path.resolve(__dirname, '..'), sub.filePath);
			lines.push(`### ${entry.name}.${sub.getterName} (${subRelative})`);
			lines.push('');
			lines.push(...subSigs);
			lines.push('');
		}
	}

	// Ensure output directory exists
	fs.mkdirSync(OUTPUT_DIR, { recursive: true });

	fs.writeFileSync(OUTPUT_PATH, lines.join('\n'));
	console.log(`POM reference written to ${OUTPUT_PATH}`);
	console.log(`POMs documented: ${entries.length}`);
}

generate();
```

## Run Plan (One-Shot Test Execution)

The `/run-plan` endpoint executes an entire test in a single HTTP call.
This is the primary endpoint for AI-driven QA -- it replaces per-step
`/pom` and `/action` calls for deterministic test scenarios.

### Request

```
POST /run-plan
{
  "title": "QA #12345: Variable appears after execution",
  "stepTimeout": 10000,
  "resetBefore": false,
  "steps": [
    {"type": "pom", "pom": "sessions", "method": "start", "args": ["python"], "timeout": 20000, "title": "Start Python"},
    {"type": "pom", "pom": "console", "method": "executeCode", "args": ["Python", "x = 42"], "title": "Run code"},
    {"type": "pom", "pom": "variables", "method": "expectVariableToBe", "args": ["x", "42"], "timeout": 5000, "title": "Check variable"}
  ]
}
```

| Field | Type | Description |
|-------|------|-------------|
| `title` | string (required) | Label for the Playwright report |
| `steps` | array (required) | Ordered steps (same schema as `/batch`) |
| `resetBefore` | boolean (default false) | Run state cleanup before steps |
| `stepTimeout` | number (default 10000) | Default timeout per step in ms |

### Response (success)

```json
{
  "passed": 3,
  "failed": 0,
  "steps": [
    {"title": "Start Python", "success": true, "duration": 2100},
    {"title": "Run code", "success": true, "duration": 800},
    {"title": "Check variable", "success": true, "duration": 400}
  ],
  "totalDuration": 3300,
  "state": { "variableCount": 1, "variableNames": ["x"], ... }
}
```

### State Reset

When `resetBefore: true`, the server runs cleanup before executing steps:
1. Dismiss overlays (3x Escape)
2. Close all editors
3. Clear notifications
4. Delete all sessions
5. Toggle bottom panel
6. Focus editor area

Each cleanup step has a 2-second timeout and fails silently.
